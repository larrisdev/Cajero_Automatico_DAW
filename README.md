# Cajero_Automatico_DAW

Aplicación web que simula el funcionamiento de un cajero automático (ATM), permitiendo realizar operaciones básicas como consulta de saldo, depósitos, retiros, pagos de servicios y visualización de historial de transacciones.

El sistema utiliza un usuario de prueba para acceder a la plataforma.
